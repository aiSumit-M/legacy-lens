# Legacy Lens 🔍

[cite_start]**Legacy Lens** ek AI-powered developer tool hai jo undocumented GitHub repositories ko instantly comprehensive aur interactive documentation suites mein transform kar deta hai[cite: 12]. [cite_start]Ye tool developers ko faster seekhne aur smarter kaam karne mein madad karta hai[cite: 8].

---

### 💡 Problem Statement
[cite_start]Build an Al-powered solution that helps people learn faster, work smarter, or become more productive while building or understanding technology[cite: 8]. [cite_start]Legacy Lens ise solve karta hai taaki jab developers project chhodein ya project purana ho jaye, toh koi bhi institutional knowledge lose na ho[cite: 14].

---

### ✨ Key Features & USP
* [cite_start]**Living Documentation:** Docs jo code commit hone par automatically real-time mein update hote hain[cite: 23, 26].
* [cite_start]**Visual Architecture Generation:** Raw code ko instantly flowcharts aur diagrams mein convert karta hai[cite: 27].
* [cite_start]**"Why" Engine:** Ek chat interface jo complex code blocks ke peeche ke historical design decisions ko explain karta hai[cite: 28].
* [cite_start]**Contextual Q&A:** Apne codebase ke saath chat karein (e.g., "Where is the user authentication logic located?")[cite: 36].
* [cite_start]**Onboarding Checklist:** Naye team members ke liye step-by-step "Getting Started" guide[cite: 34].
* [cite_start]**Legacy Risk Audit:** Purane code mein deprecated libraries aur security vulnerabilities ko scan karta hai[cite: 35].

### 📊 Comparison: Why Legacy Lens?
[cite_start]Standard **AI Copilots** (jaise GitHub Copilot) naya code likhne par focus karte hain, lekin **Legacy Lens** exclusively existing code ko samajhne par focus karta hai[cite: 19]. [cite_start]Yeh sirf line-by-line bugs fix nahi karta, balki poori repository ko analyze karke batata hai ki system as a whole kaise kaam karta hai[cite: 20].
