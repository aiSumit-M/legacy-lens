# Legacy Lens 🔍

**Legacy Lens** ek AI-powered developer tool hai jo undocumented GitHub repositories ko instantly comprehensive aur interactive documentation suites mein transform kar deta hai. Ye tool developers ko faster seekhne aur smarter kaam karne mein madad karta hai.

---

### 💡 Problem Statement
Build an AI-powered solution that helps people learn faster, work smarter, or become more productive while building or understanding technology. Legacy Lens ise solve karta hai taaki jab developers project chhodein ya project purana ho jaye, toh koi bhi institutional knowledge lose na ho.

---

### ✨ Key Features & USP
* **Living Documentation:** Docs jo code commit hone par automatically real-time mein update hote hain.
* **Visual Architecture Generation:** Raw code ko instantly flowcharts aur diagrams mein convert karta hai.
* **"Why" Engine:** Ek chat interface jo complex code blocks ke peeche ke historical design decisions ko explain karta hai.
* **Contextual Q&A:** Apne codebase ke saath chat karein (e.g., "Where is the user authentication logic located?").
* **Onboarding Checklist:** Naye team members ke liye step-by-step "Getting Started" guide.
* **Legacy Risk Audit:** Purane code mein deprecated libraries aur security vulnerabilities ko scan karta hai.

---

### 🛠 Tech Stack
* **GenAI Model:** AWS Bedrock (Claude 3 Sonnet & Titan).
* **Compute:** AWS Lambda (Serverless).
* **Storage:** Amazon S3 & DynamoDB.
* **Frontend:** React / Next.js.
* **DevOps:** GitHub Actions.

---

### 📊 Comparison: Why Legacy Lens?
Standard **AI Copilots** (jaise GitHub Copilot) naya code likhne par focus karte hain, lekin **Legacy Lens** exclusively existing code ko samajhne par focus karta hai. Yeh sirf line-by-line bugs fix nahi karta, balki poori repository ko analyze karke batata hai ki system as a whole kaise kaam karta hai.

---

### 💰 Cost Efficiency
* **Cost Model:** Pay-per-use (Serverless).
* **Estimate:** <$5.00 per 100 repositories processed.
* **Initial Development:** Free tier eligible.
